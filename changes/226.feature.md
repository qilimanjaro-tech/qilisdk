Introduced a real expression-tree AST rooted at the new `qilisdk.core.Expression` base class, replacing the previous flattened-polynomial model. Symbolic values (gate angles, Hamiltonian coefficients, schedule coefficients, model objectives/constraints) are now uniformly typed `Expression`, built from `Add`/`Mul`/`Pow`/`Constant` and the unary `Function` nodes `Sin`/`Cos`/`Exp`/`Log`/`Tan`/`Sqrt`. The new tree adds symbolic differentiation (`Expression.diff`), simplification and distribution (`Expression.simplify`/`Expression.expand`), arbitrary unary functions, and non-integer/symbolic powers, while preserving order-independent structural equality (`x + y == y + x`).
