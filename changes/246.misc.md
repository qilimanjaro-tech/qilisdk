Three new experiment types have been added:

- `TwoTonesFrequencyVsFluxQdacRampCWExperiment` — 2D two-tone spectroscopy sweeping drive LO frequency vs flux bias using a QDAC ramp waveform and continuous-wave acquisition.
- `TwoTonesPulsedSoftExperiment` — 1D two-tone spectroscopy using a soft Gaussian ring-up drive, sweeping IF frequency at a fixed flux bias point.
- `T1SoftSaturationHWLExperiment` — T1 relaxation experiment using a soft saturation drive with hardware loop unrolling, sweeping wait duration.

Each experiment has a corresponding result class (`TwoTonesFrequencyVsFluxQdacRampCWExperimentResult`, `TwoTonesPulsedSoftExperimentResult`, `T1SoftSaturationHWLExperimentResult`) that supports plotting and fitting.
