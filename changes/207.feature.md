Added an LP-format serializer (`to_lp` and `to_lp_file` in `qilisdk.utils.lp_parser`) that writes a `Model` back to CPLEX-flavor LP text, round-trippable through the existing `from_lp` parser. Also skips redundant variable-bound encoding constraints when a variable's bound matches its domain's natural extreme.

```python

from qilisdk.utils.lp_parser import from_lp_file

model = from_lp_file("knapsack.lp")
print(model)

```
