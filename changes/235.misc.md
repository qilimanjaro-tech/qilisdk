Phase plots for each experiment have been added, enabled via a new argument: `.plot(plot_type = "phase")`.

dB plots are also back (since it was only a few lines extra), which can be enabled with `plot_type = "db"`.

The default axis names have been changed from "|S21|" to "Amplitude (V)" to be more consistent.

Note that when using any plot_type other than the default "amplitude", overrides are disabled, since they are supposed to scale the outcome of the amplitude calculation, not the raw data.
